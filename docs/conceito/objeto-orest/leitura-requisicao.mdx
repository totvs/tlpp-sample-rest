Use estes métodos para ler o que o cliente enviou — corpo JSON, cabeçalhos, parâmetros na URL (`:id`) e query string (`?campo=valor`).

:::caution Objetos por referência
`getHeaderRequest`, `getQueryRequest` e `getPathParamsRequest` retornam **referência interna** — não altere o JSON retornado ([detalhes](/docs/tlpp/rest/objeto-orest/avisos-dicas#objetos-por-referencia)).
:::
