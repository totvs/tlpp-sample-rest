Para rotas dinâmicas ou legado, mapeie endpoints via JSON com `LoadURNs` — sem `@Get` / `@Post` no código.

## Quando usar

- URLs geradas em runtime
- Migração de WsRESTful
- Controle fino de `ProgramType`, `ClassName` e `Function` por verbo HTTP

## Fluxo em duas etapas

1. **Subir o serviço** — monte o `JsonObject` e chame `VdrCtrl():Start()`, incluindo o nó `LoadURNs`.
2. **Preencher `LoadURNs`** — associe path + verbo à função ou método compilado no RPO.

O TDN demonstra isso com `u_fInitService()` chamando `sLoadURNs()` antes do start.
