# Exemplos TLPP (REST)

Código-fonte em <RepoLink id="rest-samples" />, em `src/`, organizado por tema.

Esta seção é o **aprofundamento executável** da trilha conceitual do menu REST: cada página lista os arquivos `.tlpp` com link para o GitHub. Use depois de ler o conceito correspondente (ex.: [Leitura da requisição](/docs/tlpp/rest/objeto-orest/leitura-requisicao) → [oREST · leitura requisição](/docs/tlpp/rest/exemplos-repositorio/orest-leitura-requisicao)).

## Como usar

1. Leia a página conceitual no menu REST.
2. Abra o arquivo `.tlpp` pelo link da tabela.
3. Compile no RPO e adapte ao seu `appserver.ini`.

A ordem do menu espelha a documentação: primeiros passos → configurações → verbos → oREST → callbacks → auth → APIs admin.
